# Proyek Portfolio Pribadi

Halo semua! Saya ingin membagikan proyek portfolio yang saya bangun sebagai bagian dari program magang saya di **PT. Surya Citra Media**. Proyek ini merupakan pengalaman pertama saya menggunakan **Vue.js**, dan saya sangat antusias dengan apa yang berhasil saya capai.

## Tentang Proyek Ini

Ini adalah aplikasi web portofolio dinamis yang sepenuhnya saya kembangkan dari awal, dengan sistem manajemen konten (CMS) khusus yang memungkinkan saya mengelola semua aspek portofolio dengan mudah.

### Fitur Utama

#### Untuk Pengguna (Publik)

*   **Melihat Portofolio:** Pengguna dapat menjelajahi semua proyek, pengalaman kerja, sertifikasi, dan tumpukan teknologi yang saya kuasai.
*   **Memberikan Rekomendasi:** Pengguna dapat memberikan testimoni atau rekomendasi, yang akan ditampilkan di situs setelah disetujui oleh admin.
*   **Desain Responsif:** Antarmuka yang ramah pengguna dan dapat diakses di berbagai perangkat, mulai dari desktop hingga seluler.

#### Untuk Admin (Saya)

*   **Manajemen Konten (CMS):** Dasbor admin yang aman untuk mengelola semua konten di situs.
*   **CRUD Penuh:** Kemampuan untuk membuat, membaca, memperbarui, dan menghapus (CRUD) semua entri portofolio, termasuk:
    *   Proyek
    *   Pengalaman Kerja
    *   Pendidikan
    *   Tumpukan Teknologi
    *   Sertifikasi
    *   Tautan Media Sosial
*   **Moderasi Testimoni:** Admin dapat meninjau dan menyetujui testimoni yang dikirimkan oleh pengguna sebelum ditampilkan di situs.

### Struktur Proyek dan Teknologi

Proyek ini dibangun menggunakan arsitektur modern berbasis komponen, yang membuatnya mudah untuk dikelola dan dikembangkan lebih lanjut.

*   **Frontend:**
    *   **Vue.js 3:** Kerangka kerja JavaScript progresif untuk membangun antarmuka pengguna yang interaktif.
    *   **Vite:** Alat pengembangan frontend yang sangat cepat.
    *   **Tailwind CSS:** Kerangka kerja CSS yang mengutamakan utilitas untuk desain yang cepat dan kustom.
    *   **Vue Router:** Untuk perutean sisi klien dan navigasi antar halaman.
    *   **Pinia:** Untuk manajemen keadaan (state management) yang terpusat dan mudah diprediksi.
